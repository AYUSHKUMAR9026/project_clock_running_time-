# Digital Clock Project

A simple and responsive Digital Clock built using HTML, CSS, and JavaScript.

## Features

- Live running clock
- Real-time hours, minutes, and seconds
- AM/PM format
- Current date display
- Modern neon UI design
- Responsive layout

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## Project Structure

project-folder/
│
├── index.html
├── style.css
├── script.js
└── README.md

---

## How It Works

- JavaScript uses the `Date()` object to get the current time.
- `setInterval()` updates the clock every second.
- DOM manipulation is used to display the time and date on the webpage.
